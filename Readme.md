# Nautilus plugin: SWWW

This nautilus plugin will add the option "**Set SWWW Wallpaper**" to the image files, It will make a file in `~/.local/state/` called *initback.sh*
that will set the background when you run it, So when can run the swww wallpaper when boot and keep the wallpaper you left with nautilus

To install is like follows: `cp plugin.py ~/.local/share/nautilus-python/extensions/swww-wallpaper.py` and then restart nautilus with `nautilus -q`
